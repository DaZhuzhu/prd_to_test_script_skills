### TC-01 标签视图-展示标签（P0 / UI展示）

- **入口路径**：听蓝首页 → Tab页
- **前置条件**：已创建至少1个自定义标签
- **测试数据**：标签名称：工作录音、会议记录、学习笔记
- **操作步骤**：
  1. 启动听蓝App，进入听蓝首页
  2. 查看Tab页面标签展示
- **预期结果**：
  1. Tab页面展示所有标签（包含固定标签"我的录音"、"分享给我"、"收藏"及自定义标签）
  2. 标签以Tab形式横向排列展示
  3. 标签可左右滑动查看所有标签
- **可观测校验点**：Tab区域展示标签、左右滑动标签区域可查看更多标签
- **自动化建议**：适合自动化

### TC-20 标签删除-弹窗展示（P0 / UI展示）

- **入口路径**：听蓝首页 → Tab页 → 展开按钮 → 管理按钮 → 管理标签页 → 删除按钮
- **前置条件**：管理标签页已打开，有自定义标签"项目A"
- **操作步骤**：
  1. 进入管理标签页
  2. 点击自定义标签"项目A"的删除按钮
- **预期结果**：
  1. 弹出双按钮提示弹窗
  2. 弹窗标题显示"提示"
  3. 弹窗文案显示"删除后，标签中的录音不会被删除，仅删除该标签分组，是否继续？"（⚠️矛盾#4#5：文案版本不一致）
  4. 左按钮显示"取消"（黑色字体）
  5. 右按钮显示"继续"（蓝色字体）
- **可观测校验点**：弹窗标题、文案、按钮文案和颜色
- **自动化建议**：适合自动化
- **备注**：⚠️阻塞#4#5#6：弹窗文案和按钮版本不一致，需产品确认

### TC-62 添加标签-取消权限后录音删除（P0 / 数据一致性）

- **入口路径**：听蓝首页 → 分享给我Tab → 添加标签蓝信和听蓝 → 取消分享权限
- **前置条件**：分享录音已添加标签蓝信和听蓝
- **操作步骤**：
  1. 为分享录音添加标签蓝信和听蓝
  2. 取消被分享者的权限
  3. 分别查看分享给我、蓝信、听蓝标签Tab下的录音
- **预期结果**：
  1. 三个标签页下的录音都被删除
  2. 分享给我Tab下该录音消失
  3. 蓝信Tab下该录音消失
  4. 听蓝Tab下该录音消失
- **可观测校验点**：三个标签下均无该录音
- **自动化建议**：适合自动化

### TC-91 首页搜索-标题高亮（P1 / UI展示）

- **入口路径**：听蓝首页 → 搜索icon → 搜索页 → 输入关键词
- **前置条件**：有录音标题包含关键词"会议"
- **操作步骤**：
  1. 点击搜索icon进入搜索页
  2. 输入关键词"会议"
  3. 查看搜索结果中标题匹配的录音展示
- **预期结果**：
  1. 标题匹配的录音标题中关键词"会议"高亮显示
  2. 高亮标识与正常文字有明显区分
- **可观测校验点**：标题中关键词高亮、高亮效果可辨识
- **自动化建议**：适合自动化
- **备注**：⚠️含糊#7：高亮样式待确认
